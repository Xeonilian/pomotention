# 附录：术语对照

## 行动的生命周期

:::tip
过去心不可得，现在心不可得，未来心不可得。 《金刚经》
:::

一条事项从「想到」到「做完留痕」，会依次落在不同**问题域**里；活动清单里的 `activity`、计划表里的 `todo` / `schedule`、任务追踪里的 `task` ，是为了让每一层只回答一类问题，避免把「库存、日历承诺、时钟占位、执行档案、专注回合」混在同一张表里。

1. **`activity`（活动清单）**  
   回答：**仓库里有什么？**  
   事项仍处在收集、拆解、打标签、估番茄的阶段，**尚未承诺**落在计划表的哪一天；与具体执行日的绑定是可选的、可撤销的。

2. **`todo` / `schedule`（计划表）**  
   回答：**我承诺在哪一天（及何种粒度上）处理它？**  
   二者都来自活动清单中的条目，`schedule` 是与他人之间约定的事件，有跟多的“他律”，而 `todo` 可能有截止日期却其实并没有一个具体的人在事件中，更需要“自律”。

3. **`task`（任务追踪）**  
   回答：**这一轮实际执行里发生了什么？**  
   从灵感来源，发散的想法，到执行总结；能量、愉悦、打扰、书写模板等**过程数据**挂在追踪任务上；任何相关的信息。

4. **时间表与番茄时钟**  
   时间表回答：**今天 24 小时轴上，块落在哪一段？**  
   番茄时钟回答：**当前这一专注/休息回合的目标与节奏是什么？**  
   利用标度时间经验来保护时间之流的经验，去理性的看到有限，在辅助中校准感受。它们辅助你，而不是替代你的感受。

## 「计划」与「意图」的区别

:::tip
**计划**管日程与资源占位；**意图**管当下这一拍的启动与目标。
:::

为避免与日常口语混用，在**模块联动**语境下建议这样读（与产品名中的 _Intention_ 及 [什么是 Pomotention？](../intro/what-is-pomotention.md) 对齐）：

- **计划（plan）**  
  指**计划表 + 时间表**所表达的安排：把活动放进具体日期、再在当日轴上挪动时段。回答的是 **when（何时排进日程 / 轴上占位）**。

- **意图（intention）**  
  在心理学语境指 **执行意图**（Implementation Intention）：把「打算做」写成易于触发行动的条件结构；在本应用中落实为 **番茄意图**——**当前番茄（或休息）这一轮要瞄准什么**，可与计划中的条目绑定，也可以先计时再补计划。回答的是 **what / how to start（这一轮要达成什么、如何降低启动阻力）**。

## 术语表

| 英文                  | 中文     | 含义                                                                                                                                                          |
| --------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Activity              | 活动     | 活动清单里的一条记录，包括待办、预约、无所事事、内外打扰等。不是任务追踪里的「任务」。                                                                        |
| Activity Sheet        | 活动清单 | 首页右侧用来收集、筛选和管理活动的区域。                                                                                                                      |
| Energy                | 精力值   | 任务追踪里记录的当下精力感受（1–10 分），时间轴上用 ⚡ 显示；仪表盘中的「平均精力值」也来自此类记录。                                                         |
| External Interruption | 外部打扰 | 例如：来自他人来电、信息等的事项。                                                                                                                            |
| Internal Interruption | 内部打扰 | 例如：来自个人情绪、想法和感受等的事项。                                                                                                                      |
| Planner               | 计划表   | 首页中上区域：按日/周/月查看，把待办、预约等排进具体日期，管理执行与完成状态。                                                                                |
| Reward                | 愉悦值   | 任务追踪里记录的当下情绪与满意程度（1–10 分），时间轴上用 🏵️ 显示；仪表盘中的「平均愉悦值」也来自此类记录。                                                   |
| Schedule              | 预约     | 与他人约定具体时间、地点和时长的事项。                                                                                                                        |
| Task                  | 任务     | 任务追踪里当前这一轮执行的记录：描述、精力、愉悦、打扰等都记在这里；通常从活动清单里某条活动点「追踪」后产生。与日常说的「任务」或 GTD 里的任务不是同一概念。 |
| Task Tracker          | 任务追踪 | 首页中下区域：记录正在执行的任务，以及精力、愉悦、打扰与书写内容。                                                                                            |
| Timetable             | 时间表   | 首页左侧区域：在当天 24 小时轴上安排工作/休息等时间块，并把计划中的事项拖到时段里占位。                                                                       |
| Todo                  | 待办     | 用番茄块拆分的可执行项（🍅/🍒/🍇）。口语里的「待办任务」多指它，与任务追踪里的「任务」不是一回事。                                                            |
| Untaetigkeit          | 无所事事 | 与自己约定留白或发呆的时间段。                                                                                                                                |
