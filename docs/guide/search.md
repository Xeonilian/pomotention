## 1 搜索功能

- **进入搜索页面**：点击 `菜单栏` 的页面标签 `搜索` 进入搜索界面
- **输入搜索关键词**：搜索框中输入关键词，系统会在 Schedule、Todo 和 Task 数据中进行搜索
- **查看搜索结果**：返回内容包括题目、日期、对应 Task 的详情，点击左侧标题页面滚动到对应区域，点击 `+`展开内容，点击 `-`收缩内容
- **页面内搜索**：按 `Ctrl + F` 或 `CMD + F` 对页面内容进行二次搜索
