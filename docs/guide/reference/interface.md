## 页面分区与职责

- 左侧：`时间表 / 区块编辑器`，负责当天可用时间与任务占位，详见 [时间表构建](timetable.md)。
- 中上：`任务计划`，负责日/周/月/年排程与执行状态，详见 [任务计划](planner.md)。
- 中下：`任务追踪`，负责执行过程记录（能量/愉悦/打扰/文本），详见 [任务追踪](task.md)。
- 右侧：`活动清单`，负责活动收集、筛选与看板整理，详见 [活动清单](activity.md)。
- 浮动区：`番茄意图计时器`，负责专注和休息节奏，详见 [番茄意图计时器](timer.md)。

## 顶栏 Menu

### 路由入口

- `首页`：进入主工作区。
- `数据页`：查看和检索历史数据，详见 [数据查看](search.md)。
- `仪表盘`：查看统计趋势，详见 [数据趋势](./chart.md)。
- `帮助页`：查看软件帮助信息。
- `设置页`：查看状态与调试工具。
- <img src="/icons/List24Filled.svg" width="20" style="display:inline-block;vertical-align:middle;margin:0;background:rgb(244, 244, 244);border-radius: 6px;">：手机端收缩菜单按钮。
