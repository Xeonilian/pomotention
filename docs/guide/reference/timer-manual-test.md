# 番茄时钟 · 手动测试清单

::: tip
功能说明见 [番茄时钟使用说明](./timer.md)。本页供发布前或改动的 **手动验收** 使用。
:::

快速验证时可将工作时长临时设为 **1 分钟**。建议在 **Timer 独立壳** 中测试。

## 0. 测试前准备

- [ ] 记住或清空当前统计，便于对照
- [ ] 默认在 **单次模式**（🍅，非 🍕）
- [ ] 正计时相关用例：在 **设置** 中开关「结束后继续正计时」各测一遍

---

## 1. 单次模式 · 基础

| # | 操作 | 预期 |
|---|------|------|
| 1.1 | 待机点 `Wind up` | 开始倒计时，按钮为 `Squash` |
| 1.2 | 计划内点 `Squash` | 结束复位；统计 +1 条 **🥫** |
| 1.3 | 正计时 **关**，工作跑满自然结束 | 播结束音、复位；计划时长 = **开始时的设置**；统计 +1 条完成记录 |
| 1.4 | 待机点 `Break` | 休息倒计时，按钮为 `Stop` |
| 1.5 | 休息中提前 `Stop`（时长 < 计划） | 统计 +1 条休息；详情「结束方式」= **提前结束（Stop）** |
| 1.6 | 休息自然跑完 | 统计 +1 条；「结束方式」= **自然结束** |

---

## 2. 单次模式 · 正计时（设置已开启）

| # | 操作 | 预期 |
|---|------|------|
| 2.1 | 工作倒计时到 `00:00` | 不复位；左侧 `+`，从 `+00:01` 递增 |
| 2.2 | 观察 `:` | 有无 `+` 时冒号 **始终居中** |
| 2.3 | 计划内（未到 0）点 `Squash` | 仍记 **🥫** |
| 2.4 | 正计时中点 `Stop` | 按 **实际总时长** 分档 |
| 2.5 | 详情「结束方式」 | **正计时结束（Stop）**（非「提前结束」） |
| 2.6 | 休息到 0 后继续正计时再 Stop | 时长 > 计划时同上 |
| 2.7 | 正计时 **关** 时到 0 | 旧行为：自动结束复位，无 `+` |

---

## 3. 状态文案（Title）

| # | 操作 | 预期 |
|---|------|------|
| 3.1 | 开始后改 title 一次再结束 | 详情「状态文案」= 修改后 |
| 3.2 | 开始后改 title 两次再结束 | = **最后一次** |
| 3.3 | 开始后清空 title 再结束 | = `Ready to pomodoro!` |
| 3.4 | 休息中 | 显示 `Take a break`，**不可编辑** title |
| 3.5 | 有 **自定义** title 的工作段 | hover emoji 显示该文案 |

---

## 4. Pizza 连续模式

| # | 操作 | 预期 |
|---|------|------|
| 4.1 | 运行中不能切换 🍅/🍕 | 切换按钮 disabled |
| 4.2 | 序列跑完若干段后中途 Stop | 弹出「已完成的番茄/休息是否计入统计？」 |
| 4.3 | 选 **计入** | 保留已完成段 |
| 4.4 | 选 **不计入** | 撤销已完成段，仅 **1 条 🥫** |
| 4.5 | Esc / 点遮罩 | **默认计入** 并停止 |
| 4.6 | 第一段未跑完就 Stop | **不弹窗**，1 条 🥫 |
| 4.7 | Pizza 工作到 0 | **不**进入正计时 |

## 5. 统计页

| # | 操作 | 预期 |
|---|------|------|
| 5.1 | 每日行 | 日期 + 星期；`W x.x h` / `B x.x h` |
| 5.2 | emoji 网格 | 等宽；点击打开详情 |
| 5.3 | 周切换 | 上一周 / 下一周正常 |
| 5.4 | 详情字段顺序 | 类型 → 时长 → 开始 → 结束 → 状态文案 → 计划时长 → 结束方式 |
| 5.5 | 结束方式文案 | 见第 6 节表 |

---

## 6. 结束方式文案

| 场景 | 预期 |
|------|------|
| 自然到点 | 自然结束 |
| 计划内 Squash | 提前结束（Squash） |
| 休息提前 Stop（时长 < 计划） | 提前结束（Stop） |
| 正计时后 Stop（时长 > 计划） | 正计时结束（Stop） |

---

## 7. 规则 · 分档与回落

| # | 操作 | 预期 |
|---|------|------|
| 7.1 | Squash / Work / Break 三档 | checkbox **不可取消** |
| 7.2 | 取消 Long Work，跑 45min+ 工作 | emoji **回落** 🍅 |
| 7.3 | 取消 Long Break，跑长休息 | 回落 short break emoji |
| 7.4 | 改规则后回到统计页 | emoji / 折线按 **当前规则重算**（看的时候算，不改历史原始数据） |

## 9. 设置与刷新

| # | 操作 | 预期 |
|---|------|------|
| 9.1 | 运行中改 **工作/休息时长** | **下一段**生效，当前倒计时不变 |
| 9.2 | 运行中改 **正计时开关** | **到 0 时**才生效 |
| 9.3 | 设置页关白噪音（运行中） | 当前段 **通常仍响**（Pizza 面板 🔈 切换才会立刻停） |
| 9.4 | 换白噪音音轨（工作中） | **立刻**切换音轨 |
| 9.5 | F5 刷新（进行中） | 计时停；**本段无记录** |
| 9.6 | F5 后查历史 | **已完成**记录仍在 |

## 10. CSV 导出

|---|------|------|
