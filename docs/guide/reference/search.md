# 数据查看说明

## 快速导航

- 我想全文搜索历史记录：见 [基础搜索](#基础搜索)
- 我想按标签过滤：见 [标签筛选](#标签筛选)
- 我想只看重点任务：见 [星标筛选](#星标筛选)
- 我想在结果里继续编辑：见 [结果查看与编辑](#结果查看与编辑)

## 基础搜索

- 点击菜单栏 `数据` 进入搜索页面，默认按时间倒序展示。
- 在搜索框输入关键词，结果实时过滤。
- 页面内可用 `Ctrl + F` / `CMD + F` 二次检索。  
  <img src="/search.png" alt="搜索" width="600">

## 标签筛选

- 点击数据页任意可视标签，立即筛选左侧 `Activity`。
- 支持多标签叠加，采用 **AND 逻辑**。
- 再次点击已激活标签可取消该条件。
- 搜索框输入 `#` 可快速选择标签并应用筛选。
- 点击 <img src="/icons/Dismiss12Filled.svg" width="20" style="display:inline-block;vertical-align:middle;margin:0;"> 一键清除全部标签筛选。  
  <img src="/search-tag.png" alt="标签筛选" width="600">

## 星标筛选

- 点击 <img src="/icons/Star20Regular.svg" width="20" style="display:inline-block;vertical-align:middle;margin:0;border-radius: 6px;"> 仅显示已加星任务。

## 结果查看与编辑

- 点击左侧任意结果，右侧打开对应标签页显示详情（标题、描述、到期日期、标签等）。
- 点击右上角 <img src="/icons/Dismiss12Filled.svg" width="20" style="display:inline-block;vertical-align:middle;margin:0;"> 可批量关闭所有已打开标签页。
- 右侧详情区支持：
  - 编辑星标
  - 添加/删除标签（详见 [标签系统](/guide/reference/tag)）

## 限制与说明

- 数据页内容可作为后续默认 prompt 参考来源。
