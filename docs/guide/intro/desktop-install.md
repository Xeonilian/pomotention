# 桌面客户端说明

::: tip 与 PWA / 网页 的分工

- **桌面客户端**：`.msi` / `.dmg` **独立安装**；本机 **单一本地库**；**全量 JSON 导出** 仅桌面开放，详见 [账号与数据](./account-and-data.md)。置顶与系统集成以桌面端为准。
- **不打算装桌面、用浏览器即可**：安装方式、标签页与 PWA 存储差异、白屏排查等见 [PWA渐进式网页应用说明](./pwa-install.md)。

:::

## Windows

### 系统要求

- Windows 10 或更高版本
- 64 位系统

### 下载与安装

使用官方发布的 **[v0.6.4 Windows 安装包（.msi）](https://github.com/Xeonilian/pomotention/releases/download/v0.6.4/Pomotention_0.6.4_x64_en-US.msi)**。下载完成后双击运行安装程序，按向导完成安装；安装结束后从 **开始菜单** 启动应用。

::: warning 安全提示

若浏览器或系统提示「已保护你的电脑」「阻止下载」等，可在 **更多信息** → **仍要运行**，或对被拦截的下载选择 **保留**。
:::

## macOS

### 系统要求

- macOS 13.0 (Ventura) 或更高版本
- webkit 支持不完全导致 macOS 12.0 可安装，部分渲染失败

### 下载与安装

- 使用官方发布的 **[v0.6.4 macOS 安装包（.dmg）](https://github.com/Xeonilian/pomotention/releases/download/v0.6.4/Pomotention_0.6.4_x64.dmg)**。
- 下载完成后双击打开磁盘映像，将应用拖入 **应用程序** 文件夹，再从 Launchpad 或 **应用程序** 中启动。

:::warning 安全提示

首次运行时若被拦截，在 **系统设置 → 隐私与安全性**（或旧版系统中的 **系统偏好设置 → 安全性与隐私**）中按提示 **仍要打开** / 允许运行。
:::

## 自动更新

- 软件提供自动更新检测；由于网络原因可能下载失败。
- 在 **帮助页** 可查看本地与云端版本对比，并打开或关闭自动更新。

## 遇到问题

### 下载与安装

1. **下载失败**：先检查网络连接，或改用下载工具。若怀疑同步或首次启动拉取失败，可稍后再试（参见更新日志中初始化顺序与同步相关修复）。`设置页`里可对当前网络环境做连通性测试。
2. **安装报错**：确认系统版本符合要求；**Windows 7** 暂不支持。
3. **安装中断**：在 **Windows** 上可从任务管理器结束相关进程后重试安装。

### 白屏、界面长时间不加载

桌面端是 **独立应用**（不是浏览器标签页），排错与 [PWA 里的「强制刷新页面」](./pwa-install.md#pwa-blank-screen) **不是同一套操作**。

1. **先完全退出应用再重新打开**；仍异常时到 `设置页` 做网络 / 环境诊断（与 PWA 文档中的「环境诊断」为同一套能力时，说明一致）。
2. **需要重置本地数据时**：在 `设置页` 使用清除本地数据等功能。**清除前若数据未上云**，请先按 [账号与数据 · JSON 全量导出](./account-and-data.md#json-export-desktop) 导出 JSON，避免丢失。
3. 若你同时在用 **浏览器里的 PWA 或网页版**，白屏、缓存类问题以 [PWA渐进式网页应用说明 · 白屏或加载不出](./pwa-install.md#pwa-blank-screen) 为准；桌面与浏览器的数据库相互独立，除非通过登录同步或 JSON 迁移。

注册、登录、同步、JSON 导入导出（各端差异见对照表）见 **[账号与数据](./account-and-data.md)**。
