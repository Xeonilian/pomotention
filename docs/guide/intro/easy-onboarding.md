# 轻松上手

::: tip
若尚未安装，请先阅读 [安装方式总览](./install-overview.md)（桌面 / PWA / 网页）。界面分区见 [软件界面](../reference/interface.md)；各面板如何配合见 [首页区域联动](../reference/workflow.md)。
:::

## 先转起来，再细化

活动、计划、番茄、时间表、同步等模块可以穿插使用，**没有唯一正确的先后顺序**。下面只是**比较容易上手**的几种切入点，任选其一即可。

- **想先认得界面**：对照 [软件界面](../reference/interface.md) 看各分区，需要理解跨区关系时读 [首页区域联动](../reference/workflow.md)，再随便进一个模块试。
- **想先专注计时**：打开 [番茄时钟](../reference/timer.md)，选或建一个活动就开跑；计划、时间表可以之后再补。
- **想先排今天要做的**：在 [任务计划](../reference/planner.md) 里放几条，再回到番茄里执行。
- **需要多端一致**：在设置里登录；注册、同步、各端差异与 JSON 导入导出等见 [账号与数据](./account-and-data.md)。

不必一次配齐所有功能；随时在 [使用导航](../index.md) 各篇**功能参考**里查细节。
