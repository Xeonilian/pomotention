# 最小上手（五步）

::: tip
若尚未安装，请先阅读 [快速开始](./getting-started.md)（桌面 / PWA / 网页）。界面分区见 [软件界面](../reference/interface.md)。
:::

## 一条最小路径

1. 在 **活动清单** 中创建或选好活动（见 [活动清单](../reference/activity.md)）。
2. 将当日要做的活动放进 **任务计划**（日 / 周 / 月视图依习惯选择，见 [任务计划](../reference/planner.md)）。
3. 打开 **番茄时钟** 开始专注；需要时在本轮末尾或 **任务追踪** 中记录精力、打扰等（见 [番茄时钟](../reference/timer.md)、[任务追踪](../reference/task.md)）。
4. 用 **时间表** 把块时间对齐到一天里（可选，见 [日程构建](../reference/timetable.md)）。
5. 需要多端一致时在 **设置** 中登录并做好 **数据同步**（见 [数据同步](../reference/synchronize.md)）。

想按番茄工作法**分阶段**跟练，见 [番茄工作法实操总览](../pomodoro-technique/README.md)。**按钮与图标**集中速查见 [附录：按钮一览](../appendix/buttons.md)。

更细的日常节奏见 [日常工作流程](../reference/workflow.md)；全站索引见 [使用导航](../index.md)。
