# 快速使用

::: tip
若尚未安装，请先阅读 [快速开始](./getting-started.md)（桌面 / PWA / 网页）。
:::

## 一条最小路径

1. 在 **活动清单** 中创建或选好活动。
2. 将当日要做的活动放进 **任务计划**（日 / 周 / 月视图依习惯选择）。
3. 打开 **番茄时钟**，开始专注；需要时在本轮末尾或 **任务追踪** 中记录精力、打扰等。
4. 用 **时间表** 把块时间对齐到一天里（可选）。
5. 需要多端一致时在 **设置** 中登录并做好 **数据同步**。

更细的流程与习惯建议见 [日常使用工作流程](./workflow.md)；各界面与功能分区见 [使用说明总览](./index.md)。
