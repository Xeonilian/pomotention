# Pomotention 帮助文档：基于番茄工作法的实操指南

本帮助文档基于弗朗西斯科·西里洛（Francesco Cirillo）发明的番茄工作法，结合 Pomotention 软件的功能，帮助你在数字环境中实践这套时间管理系统。

> **关于原著**：如果你阅读过《番茄工作法图解》或《番茄工作法：有效地使用每一点时间和脑力》，你会发现这里的结构和书中一致。没看过书也能直接使用，看完书会理解更深。

## 快速开始

如果你现在就想试试：

1. 打开 Pomotention，找到 **ActivitySheet（活动清单）**
2. 点击添加一个任务，输入名称
3. 点击任务旁的番茄图标，开始一个 25 分钟的计时
4. 计时结束，休息 5 分钟
5. 这就是你的第一个番茄

完成后，回到文档 [00-start-here.md](00-start-here.md) 了解完整流程。
