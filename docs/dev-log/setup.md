### 帮助页面 vitepress + github pages

```
pnpm add -D vitepres
pnpm docs:dev
pnpm docs:build
pnpm docs:depoly
```

config.mts 初始页面布局

- index.md 首页内容(放在 config 里面)
- package.json

```
{
  "scripts": {
    "docs:dev": "vitepress dev",
    "docs:build": "vitepress build",
    "docs:preview": "vitepress preview",
    "deploy": "vitepress build && npx gh-pages -d .vitepress/dist"
  },
  "devDependencies": {
    "gh-pages": "^6.3.0",
    "vitepress": "^1.6.3"
  }
}

```
