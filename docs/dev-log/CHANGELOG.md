# 更新日志

## `v0.6.7` · 2026-04-21

- **新增**
  - **规划与番茄**：作废番茄交互（含双击等）

- **优化**
  - **移动端**：FAB 与选中态、视觉视口（`visualViewport`）以适配键盘顶起；任务编辑态保持、Planner 高度切换与 Home 等区域一致化。

  - **标签**：Planner 内 TagSelector 对齐与键盘/回车/手机点选行为优化。

- **修复**
  - **同步与任务**：同步后书写模板弹窗无法打开、模板复制与弹窗高度等问题。

  - **登录**：仅本地（localOnly）后登录失败、密码输入相关错误；清理无效 Supabase 配置以去除多余开机提示。

  - **活动 / FAB / 模板**：移动端输入法顶出布局、部分高度设置冲突；iPhone 任务下 FAB 勾选显示与位置；活动删除按钮与选中态等。

- **其他**
  - **帮助与指南**：更新帮助与指南站点内容——重组目录与 intro 结构、梳理页面关系，补充数据趋势与工作流程侧栏等，并新增基于《番茄工作法》原著的实践向应用指南。

## `v0.6.6` · 2026-04-10

- **新增**
  - **帮助系统**：内嵌帮助改为 VitePress 构建的指南静态资源（app-docs）；外链 Cloudflare 帮助；文档目录整理与帮助发布说明（SOP）。

  - **标签**：TagSelector 重构；新增 TagPickerPopover、Home 标签筛选弹层；搜索侧支持 `removeFilterTagId`；统一标签清除交互。

  - **同步 / 计时器**：双击触发同步；计时器快速启动。

  - **调试**：应用、更新器、同步与 Supabase 的条件化调试日志。

- **优化**
  - **启动与壳层**：精简启动路径与 `MainLayout` / `App` 结构；增加启动耗时观测（`bootTiming`）。

  - **移动端 / PWA**：manifest、主布局与 Home、规划器误触与 DayTodo 等若干交互与样式；登录失焦关闭等行为。

  - **指南站点**：VitePress 主题与搜索范围（避免搜到非帮助内容）。

- **修复**
  - **活动选中**：Tracker 同步会清空 `activeId` 时仍以 `selectedActivityId` 解析当前活动，修复状态不同步导致无法删除等问题；活动表（ActivitySheet）相关交互调整。

  - **帮助**：帮助页跳转与从帮助返回主页等路径问题。

  - **模板**：模板弹窗高度不再无限增长。

- **其他**
  - **设置**：帮助相关展示与构建说明微调。

  - **工程**：版本号与打包类型等发布修补。

## `v0.6.5` · 2026-04-02

- **新增**
  - **视口调试**：在设置中新增 viewport 调试快照功能，用于布局分析与 safe-area 调整验证。

  - **任务快捷输入**：支持任务输入的快捷输入 Markdown（md）。

- **优化**
  - **移动端 / 横屏**：引入 `visualViewport` 改善移动端高度管理；优化时间表编辑器在横屏下的图标与按钮样式，并增强 Android 横屏测试相关体验。

  - **规划器 UI**：调整日期 badge 的手机效果、周视图字体大小；优化桌面端 Home 显示周几。

  - **活动布局**：活动 section 高度铺满，改善版式密度。

  - **计时器与声音**：调整时钟恢复敏感性；移除提示音期间白噪音 duck。

  - **标签交互**：优化 TagManager 行为，回车且输入非空且搜索有结果时，按排序规则与「已选优先」选中匹配项（与列表展示顺序一致）。

- **修复**
  - **任务切换**：任务上下切换时取消 `activieId`，避免错误选中。

- **其他**
  - **文档**：更新安装帮助说明。

## `v0.5.0` · 2025-11-25

  - **显示 Schedule 内容**：在时间表界面上显示用户的 Schedule 内容，帮助用户更直观地了解自己的日程安排。

- **修复**
  - **数据关联与冗余**：
    - **Task 与 Activity 失联**：修复了 Task 与 Activity 之间关联丢失的问题，确保 Task 能够正确地关联到对应的 Activity。
    - **过去冗余数据**：清理了历史遗留的冗余数据，减少了数据存储空间的占用，并提升了数据查询效率。主要涉及对 suspend 的 todo 项，以及 tag count 维护的清理。

## `v0.4.3` · 2025-10-13

- **新增**
  - **标签筛选系统**：现在可以点击标签对左侧列表或右侧内容进行筛选，支持多个标签叠加筛选（与逻辑），并可一键清除。
  - **搜索结果标星**：在搜索结果的内容区，可以直接为条目添加或取消星标，状态会与左侧列表实时同步。
  - **搜索结果管理**：新增“一键关闭全部”功能，方便快速清空搜索结果。
  - **编辑标签**：新增搜索页面 Activity 添加删除标签。
  - **编辑标星**：新增搜索页面 Task 标星。

- **优化**
  - **UI 界面重构**：
    - 重构了任务（Task）的书写模版 UI。
    - 优化了 Planner 中任务优先级的数字显示样式。
  - **代码结构优化**：
    - 将活动（Activity）中的标签拖拽功能重构为可复用的 Composable (`useTagDrag`)，提升了代码的模块化和可维护性。
  - **性能与逻辑**：
    - 优化了任务内容（Task-Content）的获取算法，使其更加精准。
    - 统一了任务（Task）的数据来源 Activity，修正了因此产生的 Tab 计算错误。

- **修复**
  - 修复了因任务数据被存储两次，导致修改后无法正确提取的 Bug。
  - 修复了在 Mac 设备上 Emoji 跑马灯显示不正常的问题（通过增加空格）。
  - 修复了时间表（Timetable）中会错误地包含已取消（Cancelled）日程和待办的问题。
