更新日志 ​

v0.6.5 · 2026-04-02 ​

  • 新增

    • 视口调试：在设置中新增 viewport 调试快照功能，用于布局分析与 safe-area 调整验证。

    • 任务快捷输入：支持任务输入的快捷输入 Markdown（md）。

  • 优化

    • 移动端 / 横屏：引入 visualViewport 改善移动端高度管理；优化时间表编辑器在横屏下的图标与按钮样式，并增强 Android 横屏测试相关体验。

    • 规划器 UI：调整日期 badge 的手机效果、周视图字体大小；优化桌面端 Home 显示周几。

    • 活动布局：活动 section 高度铺满，改善版式密度。

    • 计时器与声音：调整时钟恢复敏感性；移除提示音期间白噪音 duck。

    • 标签交互：优化 TagManager 行为，回车且输入非空且搜索有结果时，按排序规则与「已选优先」选中匹配项（与列表展示顺序一致）。

  • 修复

    • 任务切换：任务上下切换时取消 activieId，避免错误选中。

  • 其他

    • 文档：更新安装帮助说明。

v0.6.4 · 2026-04-01 ​

  • 新增

    • 数据导入：v1–v4 历史导出格式的合并导入与闸门。

  • 优化

    • 计时器：专注序列与白噪音在多端的播放与恢复；生产环境异常与「打开设置页后失效」类问题排查后的修复；清理临时调试日志。
    • PWA：更新流程与更新提示的稳定化。
    • 发布前：UI 与其余小调整。

  • 修复

    • 登录：macOS 下无法登录等问题。
    • Timer：动画与部分环境上的表现问题。

v0.6.2 · 2026-03-25 ​

  • 新增

    • 移动端任务与 FAB：底部/浮标与按钮体系统一（显隐、repeat、互斥、双击等）。

  • 优化

    • 移动端：任务编辑收尾、规划器交互、标签与课表相关的一批修复。

v0.6.0 · 2026-03-19 ​

  • 新增

    • 移动端与 PWA：手机端视图切换、安装与布局适配；计时器可视化；上传侧防抖。
    • 规划与活动：子活动/子任务；周计划时间轴；年视图下标签颜色与交叠显示规则；新块可自动落到当前时间之后；多类 rank 支持取消排序。
    • 筛选：基于标签的筛选扩展至年度等视图。
    • 数据与账号：登出时保存策略；帮助与本地说明；移除 WebDAV，保留文件导出；可选跳过登录等新流程。
    • 认证布局：PWA 登出与 MainLayout 路径对齐。

  • 优化

    • 同步：机制修补、时间戳策略、初始化顺序（降低首启下载失败概率）。
    • Timetable / Timeblock：拖动情境下全局索引与类型匹配逻辑统一（isValidGlobalIndex、isNextSegCategoryMatch、isCategoryMatch 等）。

  • 修复

    • 桌面端：置顶（ontop）、触摸、换机白屏、关机锁死类问题；Timeblock 触摸与部分界面映射。
    • Activity：父子折叠、帮助页恢复与 Planner 相关 UI 收敛；部分不同步场景。
    • 标签：展示与 Banner。
    • UI：Timetable 按钮与美化；手机与桌面多轮界面不适应与逻辑问题。

v0.5.2 · 2025-12-02 ​

  • 修复

    • 窗口与交互：置顶（ontop）、设备触摸；Timeblock 触摸与 UI 地图恢复。
    • 同步：同步机制一轮全面修补。
    • 数据路径：PWA 登出适配 MainLayout；拖动相关崩溃与白屏。

  • 优化

    • 启动：调整应用初始化顺序，降低初始化阶段拉取失败概率。

v0.5.0 · 2025-11-25 ​

  • 新增

    • 网页版：数据多端同步与随时最新版本。
      • Supabase 数据库服务：引入 Supabase 作为后端数据库服务，实现数据的云端存储和同步。
      • 多端数据同步：支持多设备之间的数据同步，保证用户在不同设备上使用时数据的一致性。解决了数据孤岛问题，并确保用户始终使用最新版本的数据。
      • 访问地址：https://pomotention.pages.dev
    • Timetable（时间表）：
    • 显示 Schedule 内容：在时间表界面上显示用户的 Schedule 内容，帮助用户更直观地了解自己的日程安排。

  • 修复

    • 数据关联与冗余：
      • Task 与 Activity 失联：修复了 Task 与 Activity 之间关联丢失的问题，确保 Task 能够正确地关联到对应的 Activity。
      • 过去冗余数据：清理了历史遗留的冗余数据，减少了数据存储空间的占用，并提升了数据查询效率。主要涉及对 suspend 的 todo 项，以及 tag count 维护的清理。

v0.4.3 · 2025-10-13 ​

  • 新增

    • 标签筛选系统：现在可以点击标签对左侧列表或右侧内容进行筛选，支持多个标签叠加筛选（与逻辑），并可一键清除。
    • 搜索结果标星：在搜索结果的内容区，可以直接为条目添加或取消星标，状态会与左侧列表实时同步。
    • 搜索结果管理：新增“一键关闭全部”功能，方便快速清空搜索结果。
    • 编辑标签：新增搜索页面 Activity 添加删除标签。
    • 编辑标星：新增搜索页面 Task 标星。

  • 优化

    • UI 界面重构：
      • 重构了任务（Task）的书写模版 UI。
      • 优化了 Planner 中任务优先级的数字显示样式。
    • 代码结构优化：
      • 将活动（Activity）中的标签拖拽功能重构为可复用的 Composable (useTagDrag)，提升了代码的模块化和可维护性。
    • 性能与逻辑：
      • 优化了任务内容（Task-Content）的获取算法，使其更加精准。
      • 统一了任务（Task）的数据来源 Activity，修正了因此产生的 Tab 计算错误。

  • 修复

    • 修复了因任务数据被存储两次，导致修改后无法正确提取的 Bug。
    • 修复了在 Mac 设备上 Emoji 跑马灯显示不正常的问题（通过增加空格）。
    • 修复了时间表（Timetable）中会错误地包含已取消（Cancelled）日程和待办的问题。

v0.4.2 · 2025-09-30 ​

  • 新增

    • AI 对话功能初版发布，支持 Markdown 渲染。
    • 内置 API 密钥，并发出支持开发号召。
    • Planner 计划模块新增跑马灯与自定义提示。

  • 优化

    • 重构了多个核心模块，包括 Planner、Task 和 Timetable，以解耦和提升可维护性。
    • 统一并调整了多个 UI 界面，包括 AI 对话布局、搜索返回界面、Timetable 优先级配色等。
    • 提升了应用状态的记忆能力，例如主页和 Planner 的选中状态。

  • 修复

    • 修正了同步时出现的 emoji 格式错误。
    • 临时修正了计时器进度条长度不足的问题。
