PWA 快速安装 ​

本文档管什么

只管：打开 https://pomotention.pages.dev、安装 PWA、白屏 / 加载慢、登录或离线、JSON 导入导出、卸载与清数据、第一次点哪里。

不管：下载 .msi / .dmg、Windows/macOS 系统要求、桌面端 自动更新开关 —— 这些在 快速开始（桌面客户端）。

目标：5 分钟内完成「能打开 → 可选装 PWA → 可选导入数据 → 第一次记录」
与安装包版本对齐：当前文档撰写时对应 v0.6.4（功能以线上为准）

1. 推荐使用方式（PWA） ​

  • 网页版：新版本在站点发布后，一般刷新页面即可；若界面异常可尝试强制刷新或清除该站点的浏览数据（与浏览器实现有关）。 最推荐：使用浏览器将网页安装为 PWA（Progressive Web App），再以独立应用形式打开。

  • 访问：https://pomotention.pages.dev

  • 支持 Service Worker 离线（在无网或弱网下仍可能打开已缓存的页面）

  • 数据主要保存在本地（如 localStorage 等），与是否登录云端无关

2. 安装步骤（按平台） ​

电脑：Edge / Chrome / Safari ​

  1. 浏览器打开 https://pomotention.pages.dev
  2. 在地址栏或菜单中找到 「安装应用」 / 「安装 Pomotention」 等入口并完成安装

常见差异：

  • Windows（Edge / Chrome）：通常会明确询问「是否安装此应用」
  • macOS 14+（Safari）：支持直接安装并保存到 Dock
  • macOS 13.7.8（实测）：Safari 入口能力受限，建议安装 Edge 后使用其 PWA 安装入口

已验证环境：macOS 13.7.8 可通过 Edge 安装 PWA；更高版本请以浏览器当前提示为准。

手机 ​

  • iPhone / iPad（Safari）：分享 → 添加到主屏幕
  • Android（Chrome 等）：菜单 → 添加到主屏幕 或 安装应用

未在文档中单独列出的系统，只要浏览器支持 PWA，理论上也可用类似方式安装，以浏览器实际提示为准。

3. 页面白屏或一直加载不出来 ​

若遇到 白屏、长时间空白、内容不出现，可按顺序尝试：

  1. 普通刷新：F5 / 刷新按钮；mac 上常用 Command + R
  2. 强制刷新（绕过缓存）：Command + Shift + R（mac）或 Ctrl + Shift + R（Windows）
  3. 清空本站缓存后再打开：在浏览器设置中清除 本站数据 或 缓存（你遇到的 mac 白屏，在清空缓存后恢复，即属此类）
  4. 若使用的是 从图标打开的 PWA，可暂时用 浏览器标签页 打开同一网址对比是否正常，以区分是否为安装实例的缓存问题

网络受限场景（需要代理） ​

有用户反馈：不翻墙打不开，翻墙后可打开。这通常说明首次资源请求被网络拦截或超时。

  • 建议先在可访问网络下（例如开启代理）完成一次完整打开
  • 等页面加载完成后，再按正常方式安装/打开 PWA
  • 首次缓存成功后，后续在离线或弱网场景下通常仍可打开已缓存内容（受缓存是否完整影响）

PWA 与「普通网页标签」不是同一份存储

在多数浏览器里，同一网址在「仅标签页打开」与「已安装的 PWA」 可能对应 不同的站点存储分区（例如 localStorage 等不互通）。
因此：换了一种打开方式，可能像「另一份本地数据」；固定一种主要使用方式（推荐已安装的 PWA）可减少困惑。

4. 登录、注册失败与离线使用 ​

  • 能注册 / 登录：可选用云端同步（依赖与 Supabase 的连接）。
  • 无法注册或连不上服务器：仍可使用 离线模式；核心数据会落在 本机。

具体按钮文案以当前版本界面为准（例如「跳过」「离线使用」等）。

5. 数据导出与导入（搬家） ​

格式为 JSON，导入导出在测试流程中已通过验证。

老版本迁移说明（v0.4.4） ​

  • 在 v0.4.4 中，可进入 帮助页，使用 数据同步 按钮执行全量数据导出
  • 导出的 JSON 文件可直接用于新版本的数据迁移

导出（备份）：

  1. 打开 设置
  2. 使用 数据导出，保存得到的 JSON 文件（请自行保管好文件）

导入：

  1. 打开 设置 → 数据导入
  2. 选择此前导出的 JSON 文件并按提示完成

导入前建议先导出当前数据

若当前设备上已有数据，导入可能覆盖或与现有数据合并，行为以软件内提示为准；重要数据请先 导出备份。

6. 卸载、清除本地数据 ​

卸载 PWA / 从主屏幕移除 ​

  • Windows：对已安装的应用图标使用右键 → 卸载（或系统/浏览器提供的卸载入口）
  • macOS：从程序坞或启动台移除应用（与浏览器/PWA 实现有关，以系统提示为准）
  • 手机：长按主屏幕图标 → 删除应用 / 移除

完全删除本地数据并重新开始 ​

  1. 在应用内打开 设置
  2. 使用 清除所有本地数据 / 重置 类功能（名称以界面为准）
  3. 确认前请 务必先导出 需要保留的 JSON

若仅想解决缓存导致的显示异常，优先使用 第 3 节 的刷新与清缓存，不一定需要清空全部数据。

7. 第一次使用（功能层面，约 2 分钟） ​

  1. 创建或选择一个 活动
  2. 把它放进 今日 / 当日计划（以你界面中的模块为准）
  3. 启动 番茄计时，完成一小段专注
  4. 按需记录状态（如精力、愉悦等，若当前版本提供）

完成后可继续在侧边栏 「使用说明」 中按模块深入阅读。
