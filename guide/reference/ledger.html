收支账本使用方法 ​

TIP

本软件 不是专业记账工具：没有余额、对账、多账套。目标是在计划表标题里顺手记下几笔，方便回看与统计——这就是内置的收支账本。

快速导航 ​

  • 我想在标题里记一笔：见 在标题里记账
  • 我想看某一天某条的明细：见 查看单条明细
  • 我想看一段时间的汇总：见 收支统计
  • 我想改 / 删 / 追加：见 在统计里改删追加
  • 我想用标签当分类：见 分类标签
  • 我想确认边界：见 限制与说明

常用按钮 ​

按钮图标功能说明
计划表顶栏：打开收支统计；待办 / 预约行上：打开该条已录入明细
统计弹窗内：展开 / 收起书写帮助
删除一笔（软删；标题汇总会尽量同步）
统计明细最下行：追加一笔（不经过标题语法）

在标题里记账 ​

适合在日视图编辑待办或预约的标题时顺手记。

  1. 编辑标题，写入金额段（见下表语法）。
  2. 用结尾符表达「这一段要入账」：￥ / ¥ / $ / %（手机键盘常用 %）。
  3. 退出编辑（失焦或回车）后才会解析入账；仅输入结尾符当下不会单独入账。
  4. 入账成功后，标题会变成「日记正文 + 汇总括号」，例如 买菜 西瓜（-55）。

书写语法 ​

写法含义
-30 / +500（空格后）或行首 -30 / +500开始一笔；默认支出，+ 为收入
￥ ¥ $ %（及全角 ＄ ％）记账段结束；没有结尾符则整段不入账
; / ； 或段内再次出现 -/+金额同段多笔分隔
#分类名写在记账段内时作为收支分类（不是活动标签，见 分类标签）

金额为正数，最多两位小数。金额后须接空格、#、;、结尾符或段尾，避免把 -3度 误当成记账。

示例 ​

保存前（编辑框）保存后标题（示意）
买菜 -30 西瓜 -25#菜市 喝的￥买菜 西瓜 喝的（-55）
奖金 +5000; 午饭 -30%奖金 午饭（-30 +5000）
-10 早餐￥早餐（-10）
开会 -30买菜（无结尾符）不入账，原文保留
今天 -3度出门（无结尾符）不入账

TIP

汇总括号里的数字是整数、且只显示非零项。再次编辑时会先剥掉旧括号，再按当前标题解析。

查看单条明细 ​

  • 当某条待办或预约已有未删除的收支时，行上会出现 。
  • 点击可查看该条关联的笔数、分类、备注与金额；可删除单笔。
  • 删除后会尽量从标题正文去掉对应备注，并重写汇总括号；若你已大幅改过标题对不上，则可能只更新括号。

收支统计 ​

  1. 在计划表顶栏点击 （标题「收支统计」）。
  2. 弹窗内可见：笔数、支出、收入、结余；支出分类饼图；收支趋势；明细表。
  3. 明细表头可切换排序：
    • 日期（时序）
    • 分类
    • 金额（从多到少）
  4. 在日 / 周视图下，可点击「收支趋势」中的日期，切换本软件当前日期（便于对照某一天）。
  5. 需要语法提示时，点明细旁的 。

在统计里改删追加 ​

操作做法
改在明细行直接改分类、备注、金额；失焦或回车生效。来自标题的行改金额后，对应标题汇总括号会更新。
删行首
追加最下行点 ，填金额后失焦或回车；分类可选、备注可选。追加日期与 本软件当前选中日期一致。

分类标签 ​

  • 在记账段内输入 # 并选择标签 → 写入该笔的分类，不会变成活动上的标签。
  • 在结尾符之后或标题其它位置输入 # → 仍是普通 活动标签。
  • 统计弹窗里点分类列，也可为该笔选择 / 新建分类标签。

示例：

  • 买菜 -30#菜 → 选「菜市」→ 分类归到菜市
  • 买菜 -30￥ #紧急 → #紧急 在结尾符后 → 活动标签

限制与说明 ​

  • 必须有结尾符才入账；没有 ￥ / $ / % 等时，全文当普通标题。
  • 同一标题多次保存：有新记账段则追加新行；没有新段时不删旧行，仍会重写汇总括号。
  • 登录账号后，收支可随云同步保存；清本地后重新登录，已同步数据应仍可见（依赖网络与账号状态，见 账号与数据）。
  • 不做余额、对账、报表导出；复杂账本请用外部工具。
