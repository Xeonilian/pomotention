# Pomotention - 番茄意图

🍅 **基于番茄工作法与执行意图的自我照顾系统**

- [快速开始](https://xeonilian.github.io/pomotention/)

### 🚀 主要功能

- **日程管理** - 创建工作/娱乐模板，智能计算可用番茄时间
- **活动管理** - 创建与筛选任务、待办、无所事事等多类型活动
- **番茄时钟** - 完整计时控制，自动记录，自定义专注/休息循环
- **执行追踪** - 预估 vs 实际误差分析，打扰事件记录，精力值管理
- **反思记录** - 任务关联的深度思考和总结
- **今日待办** - 自动提取当日计划，任务流转
- **数据可视化** - 时间轴分布、番茄统计、历史数据分析
